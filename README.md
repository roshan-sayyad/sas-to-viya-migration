# SAS to SAS Viya Migration Framework

## Overview

Enterprise-style migration framework demonstrating modernization of legacy SAS workloads into SAS Viya using orchestration, automation, and validation frameworks.

---

## Architecture

Legacy SAS → Migration Framework → SAS Viya → Validation

---

## Components

### Migration Automation
- Python job scanning
- SAS inventory analysis

### Orchestration
- Apache Airflow DAGs
- Batch scheduling concepts

### Validation Framework
- Row count validation
- Aggregate reconciliation
- KPI parity checks

### Configuration Management
- YAML-driven migration controls
